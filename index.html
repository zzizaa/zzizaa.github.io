Samurai Shiba

DEMO

Samurai Shiba is a game developed for the Videogame Design and Programming course at the Politecnico di Milano.
The game was developed using Unity, marking my first experience with this game development engine. This project not only enhanced my technical skills but also deepened my understanding of game design principles.
Samurai Shiba is a 2D platform game set in a world shrouded in darkness. The protagonist, Shiba, awakens with no memory in a desolate landscape. He is guided by the Golden Shadows, mysterious entities that point him toward his destiny. Shiba's journey is filled with dangers and strange creatures.
At the end of each zone, Shiba faces a powerful boss. Defeating these adversaries is crucial not only for progressing in the game but also for freeing his friends trapped in darkness and regaining his lost abilities.
Our hero will travel across multiple dungeons with a lot of strange creatures and evil powers.

2019: A Space Odyssey

2019: A Space Odyssey is a 2D platform game that I developed in collaboration with another student, who handled the art, during the Videogames Course at Università degli Studi di Udine. Interactive Showroom

For my Bachelor's Degree thesis, "Interactive Showroom," I developed an application using Unreal Engine 4 and AWS services to host a showroom on a website. This application features a showroom for interior design elements, showcasing high-poly models created by my tutor.
The application was built using Unreal Engine 4 and hosted on AWS EC2 instances, which provided the necessary GPU resources for rendering the high-quality models. We utilized the Pixel Streaming plugin from Unreal Engine to enable interaction between the web page and the application. This allowed us to capture user input from the web and apply it within the showroom environment. I employed HTML, CSS, and JavaScript to customize the basic site provided by the Pixel Streaming plugin, ensuring a seamless and interactive user experience.
The showroom features detailed and high-quality interior design models, enhancing the visual appeal and realism of the application. By leveraging AWS EC2 instances, users could connect to the showroom from various devices, ensuring broad accessibility and performance.

Tools used:

  • Amazon AWS
  • HTML
  • CSS
  • JavaScript

Multiplayer FPS

One of the most significant projects I've worked on during my learning journey was a demo of a strategic shooter game, similar to CS:GO. This project served as a foundation for improving my understanding of Blueprints and programming principles within Unreal Engine 4.
The game is a first-person shooter (FPS) where precise aiming is crucial. Each weapon in the game was programmed to handle recoil and shooting mechanics differently. I developed a system where bullets spread more when moving, emphasizing the need for players to stand still for more accurate shots, adding a layer of tactical depth.
A key component of this project was learning to implement multiplayer functionality using Blueprints. I created a system that allowed multiple players to join and compete in the same game session, utilizing Unreal Engine's networking capabilities. This process helped me gain a deeper understanding of multiplayer networking, data synchronization between players, and common multiplayer challenges such as latency and maintaining a smooth experience for all users.

StreaweAR

VIDEO

StreatweAR is an innovative mobile application that I developed during the Developing App for Mobile course at Università degli Studi di Milano. This iOS app was created using XCode, Swift, SwiftUI, and ARKit.
Inspired by the gameplay mechanics of Pokémon GO, StreatweAR allows users to discover and collect virtual clothing items instead of Pokémon. Users can locate chests on the map, collect unique clothing items, and view them in augmented reality.
The core concept is based on the idea of creating a collection of unique, NFT-based collectibles, ensuring that each clothing item is one-of-a-kind and can be seen by others through the app's camera functionality.
Although the NFT implementation has not yet been realized, it represents a potential future development for the project. StreatweAR merges fashion with cutting-edge AR technology, offering an engaging and interactive user experience.

Interior Design

VIDEO

For my Master's thesis in Computer Science, I developed an innovative application leveraging the passthrough feature of the Oculus Quest 2 to transform it into an augmented reality headset.
This application allows users to position virtual furniture elements within a real-world room. The process begins with a mapping phase where the user identifies the intersection points between the floor and walls, as well as between the walls and ceiling. After specifying the corners, a mesh of the room is generated. The user can then interact with this mesh using ray casting and place virtual models within the space. Hand tracking is utilized for managing interactions. With hand pose detection, users can switch between different modes: Build Mode, Edit Mode, and Grab Mode.
The mode can be identified by the color of the user's hands.

  • Build Mode: An interface on the user's left hand displays a catalog of models. From here, users can select models, choose materials, and instantiate them into the scene. The right hand's ray cast allows for interaction with the instantiated models.
  • Grab Mode: Users can interact directly with the models, moving them with one hand or rotating them with both hands.
  • Edit Mode: This mode also features an interface on the left hand, which is used to apply modifications to selected models. Models are selected using the ray cast from the right hand. Users can modify existing models in the scene and, if necessary, delete them.
Hand pose detection uses three specific gestures of the left hand to change modes. This application merges cutting-edge AR technology with interior design, offering a highly interactive and immersive user experience.

Tools used:

  • Unity
  • GitHub

Signs of the Elements

  • TRAILER
  • GAMEPLAY

In collaboration with another developer, we created "Signs of the Elements" a virtual reality game designed for the Meta Quest 2 using Unity.
This project represented both a technical and creative challenge, offering players an immersive experience as a powerful mage who can control the four elements: earth, fire, air, and water. In the game, the player takes on the role of a mage who, during an arcane experiment, accidentally opens a portal that transports him into a dark dungeon filled with monsters. Armed only with his elemental powers and a sword, the player must fight through hordes of enemies to survive and find a way to escape.
The player’s left hand is dedicated to controlling the elements, allowing them to cast powerful spells such as fireballs, air vortices, earth shields, and water jets to defeat enemies. The right hand wields a sword, enabling the player to deliver finishing blows to enemies weakened by their magic.
To escape the dungeon, the player must activate four shrines, each tied to one of the elements. Only after activating all the altars will the portal reopen, allowing the mage to return to his world.

Tools used:

  • Unity
  • Github

Roma Turris VR

In this project, I developed a story-driven virtual reality experience in Unity using the XR Interaction Toolkit, focused on environmental storytelling and immersive exploration. The experience takes place in a degraded urban neighborhood, where the player explores the environment and encounters symbolic elements such as murals, roses, and bees.
A mysterious hooded figure guides the player toward a hidden reality, leading to a transition into a parallel historical dimension that reveals the past of the same place through narrative interactions and exploration. Gameplay is centered around guided VR interactions, narrative events, and world-space UI, with smooth scene transitions and comfort-aware locomotion.
In the final phase, the player returns to the original environment, now transformed in color and atmosphere, representing renewal and a new perspective on the neighborhood.

Tools used:

  • Unity
  • Github

Room of innovation

This project is a prototype developed with Unity and the XR Interaction Toolkit, designed as a cutting-edge room concept for studios.
The goal was to create an immersive virtual environment where users could explore various aspects of television production in an innovative and interactive way. The virtual room offers the ability to manage a television schedule and also allows users to dive deeper into the role of a television host.
This prototype was created to enhance the interaction between television professionals and technology, proposing a more dynamic and engaging work environment.
