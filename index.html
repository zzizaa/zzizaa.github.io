Interactive Showroom

For my Bechelor Degree thesis "Interactive Showroom" I develope an application using Unreal Engine 4 and the AWS service to host a showroom on a website. The application consist in a Showroom for Interior Design elements that are high-poly models created by my tutor. We used Pixel Streaming, an Unreal Enigne plugin to connect the project with the webpage to have the possibility to get the input from the web work into the application. For the host we used AWS EC2 Instances that provide GPU for rendering all the meshes and give to possibility to connect to the showroom from all kind of devices. I used HTML, CSS and JavaScript to make the necesary changes to the basic site provided by the Pixel Streaming plugin.

Tools used:

  • Unity
  • Amazon AWS

StreaweAR

VIDEO

StreatweAR is an application developed during the Developing App for Mobile course at Università degli Studi di Milano. I develope the app for iOS using XCode, Swift, Swift UI and ARKit. The application is similar to Pokemon GO but with Collectibles for clothes instead of Pokemon. The user can find Chest on the Map and can collect them and wear them using Augmented Reality. The base idea is to implement a Collection based on NFT to have unique collectibles for unique clothes that also other people can see on you through the camera of the App.

Tools used:

  • Swift
  • Apple
